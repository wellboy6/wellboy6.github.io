莫问收获但问耕耘

发表于 2020-09-17 | 分类于 阅读

所谓成事，需强壮的身体，需强大的内心，需耐得住寂寞，需莫问前程。

出生于普通人之家，从小的家庭教育和学校教育都很普通。精神与外在的成长都需要自己慢慢探索，成长为一个完整的人，这是本科毕业之前的教育没有给我的，也是家庭无法给我的。或许之前也有怨言，但就当下而言，我已坦然承认并接受这一点；也因此这些年在蹉跎中不断调整自己，使自己人格、习惯、思想不至于滑入黑暗之中。

毕业之初，内心完完全全的空白，犹如稚童直接面对这一个酱缸般的社会，种种不适，精神与外在均萎靡不振、精气神不再。这五年来，困顿于此，困顿于没有答案、没有路径，不得不去不断追寻自己的答案，中间种种蹉跎，都未至于完全放弃希望与光。甚至还在期间戒掉了烟瘾，戒过才发现成就感并不大，依靠外在或某一件事给自己安全感太过艰难，一切都是过程、而非结果。

凡是种种，或游戏或沉沦或读书或奋进，或逃避，终于前后花费五年时间，从二十三岁到二十八岁，人格部分稍稍健全完整，我敢说做人部分已经不再困扰于我，纵使本身依然有诸多缺陷弱点，也不至陷于自怨自艾、自暴自弃，只待找到合适机会慢慢调整。至于抑郁等负面情绪，伴随二十八年的成长，也可说勉力与之相安共处，坏处不过是难以天天做到「慎独」。

做人一面无法困顿于我，当下与过去便无法困顿于我。而当下与未来于我而言，便剩下「做事/成事」一事而已。蹉跎了五年终于在做人方面建立了合适的自信、或叫自洽；带着拥有缺点的自洽，也终于要面对庸庸大众所必须面对的问题，要做何事以支撑自己去不断完善自己做人的修炼。或言之，不再对做人这一无答案之事困惑之后，要面对的便是，「如何做事」这一有答案的问题，既有答案，便不至于不知所措。

转眼已经返回郑州四个半月时间，138天，一年的三分之一强。落地伴随着阵痛，以及不可知的困惑与挫折。然时间终究会帮助人度过并接受现实。近日面临较大的挫折，纵然不害怕不恐惧，也不得不找些事情让自己有些事做，不至于陷入小情绪的感受中无法挣脱。选了冯唐的《成事》，阴差阳错，倒也选的挺对。看完本篇，方才知道「莫问收获、但问耕耘」是曾文正的家训，很好，梁启超也将这话放到家书里写给了孩子，很好。

无论做人与做事，有些天才一眼能看穿本质，不断奋进，如高领张磊、施一公；有些人如我，普普通通，只能靠时间、靠思考（断无顿悟可言）、靠种种难捱之事慢慢磨练，才能稍稍有些感悟。还好，不晚，人生就算只得六十年，也还有三十多年好活，普通人行尽至此，不后悔不贪恋。尽力而为，尽力直面，争取三十而立四十不惑。

人生不如意十之八九，人之不快乐也十之八九，这世界不完美之事也是十之八九。二零一零年代，我的黄金时代，已然过去了。

以上，是读冯唐讲曾文正的一些想法。

老天爷不喜欢三种东西：巧、盈、二。取巧、自满、不定。

什么是二：猜疑心、不忠心、无恒心。

想过好一生，也不复杂，不做老天不喜欢的事，不取巧、不自满、不犹豫，不二。

工作狂人的座右铭。

  1. 第一，没有舒适区域，大家都不容易。诗和远方在天外。
  2. 第二，时间不是用来挥霍的，是用来干活的。

天下事一一责报，则必有大失望之时。

  1. 做事求回报，肯定要失望。
  2. 有回报的，十件事没有一两件，按这个比例去预期，不容易失望。
    ——感悟：做事非不求回报，而是回报太难，不可强求。莫问收获，但问耕耘。

天下事未有不从艰苦中得来而可久可大者也。

  1. 容易的事，都是小事。
  2. 大事、长远的规划，都难做。
  3. 难做的大事，艰难地慢慢做，一定有可以做成的那一天。
  4. 不难，不做，永远不成。

与人为善：

  1. 与人打交道，不管给予，还是获取，要始终秉承一个善意、好心。
  2. 但是，口花花的所谓“为你好”，不是“善”，切记。
  3. 要有行动，要有给予，给钱、给资源、给时间，是真正的给予，反之亦然，“彼此以善相浸灌”。
  4. 我有几个老哥，每几个月，我烦了的时候，总是找他们去喝酒。我去了他们就开心，我喝嗨了他们就全劝我早些走，去成就那些我想成就的事儿。十几年下来，我有一天忽然问自己，他们为什么事开心？他们要成就什么？我竟然一无所知。

见识：

  1. 见识越多，越能指导自己的小，越能把自己放到合适的位置。
  2. 个人永远渺小，个人永远是过客，成事的人最多是个协调者。
  3. 天大，如果对天有意见，那就等天定一定；地大，如果对地有意见，那就等地静一静；书多，那就等别人和我讲讲我没看过的书；事儿杂，那就和其它能干的人一起去处理它。

带团队的智慧。

  1. 把我的知识、技能多教给几个人。
  2. 用他人的长处补足我的不足。
    ——人类也是这样用电脑、AI、大数据和云计算的。

带团队不能懒，一懒生百病——

  1. 纪律松懈
  2. 管理不严格
  3. 临战、做事迟钝，没有战斗力

  4. 一处迟钝，拖累整个项目组
    ——要求自己和团队和他人，从早起开始，继以准时。

  5. 不能痛哭长夜的人，不足以语人生。

  6. 不能早起的人，不足以托重任。
  7. 不能控制自己时间的人，答应之后无故爽约的人，绝对不是能帮你成事的人，从你的人生中删除吧。

欢喜，息心养性：

  1. 曾国藩的欢喜三经街：累死，能酣睡；看开，能笑忘；高声朗读，声若金石。
  2. 人间有三个空间能息心养性：重症病房、山房、书房。
  3. 病房、山房、书房，都是心房，去待待，去打开，那些房间能打开多大，你的心胸就有多大。

恒心：

  1. 坚持就是胜利，做起来很难。
  2. 难，不意味着不做，所以要天天讲、月月讲，尽力去做。
  3. 屡败屡战和屡战屡败不一样。一个是还有胜算，甚至一定能胜，一个是一定败，一定再败。

能吃苦：成事的人，暂时没了成事的环境，并不意味着放任自己，而是坚持锻炼，保持体重，保持体能，拼命读书、思考。

戒烟难，戒别的也难。
——感悟：但还是要戒。

古人办事，掣肘之处，拂逆之端，世世有之，人人不免。恶其拂逆而必欲顺从，设法以诛锄异己者，权奸之行径也。听其拂逆而动心忍性，委曲求全，且以无敌国外患而亡为虑者，圣贤之用心也。借人之拂逆，以磨砺我之德性，其庶几乎！

  1. 既然做事，就有掣肘、不顺心。握越大权，做越大事，更是如此。
  2. “借人之拂逆，以磨砺我之德性”，都是修行。
  3. 总想着，“逆我者必诛”，是小孩子打急了，或者是暴君等不及了。暴君敢杀，敢杀无赦，国家就敢“无敌国外患而亡”。
    ——【感悟】德性、修行：这两点都是我所欠缺的，遇到事情太容易急躁了，现在开始改变自己还不晚，加油。

为政之道，得人、治事二者并重。得人不外四事：曰广收、慎用、勤教、严绳。治事不外四端：曰经分、纶合、详思、约守。

  1. 得人——建团队。广泛招人、谨慎使用、辛勤教导、严格管理。曾国藩讲的和MBA教的一样：人才管理四方面，选、用、育、留。
  2. 治事——做项目。尽调分析、通盘斟酌、规划周全、执行坚决。
  3. 一个没想透、没想好的战略，如果坚决执行，会死人，会误事。
  4. 一个想透了、想好了的战略，如果不坚决执行，会死人，会误事。所以，每一步，都要朴拙勤慎。

每日须以精心果力独造幽奥，直凑单微，以求进境。一日无进境，则日日渐退矣。

  1. 曾国藩的“修行”，每天要进步。辛苦！
  2. 关于人性，关于集体人性，关于中国历史智慧，有些见识，要日日修，日日挖掘。总觉得没真明白，总觉得自己偷懒了。

清单：

  1. 于清早单开本日应了之事，本日必了之。
  2. 自己给自己买个本子，每天早上列出每天必干的三件事，每天上床前检查，至少干完第一件。

与胡中丞熟商江南军事。胡公论及凡事皆须精神贯注，心有二用，则必不能有成。余亦言军事不日进则日退。二人者皆许为知言。

  1. 一心一用。
  2. 一日一进。
  3. 更多的现世人，哪怕吃一个饭、开一个会，都不能不时常看他的手机，似乎有天大的事儿找他。

文集

慎独论：独也者，君子与小人共焉者也。小人以其为独，而生一念之妄，积妄生肆，而欺人之事成。君子懔其为独而生一念之诚，积诚为慎，而自慊之功密。彼小人者，一善当前，幸人之莫我察也，则趋焉而不决。一不善当前，幸人之莫或伺也，则去之而不力。幽独之中，情伪斯出，所谓欺也。惟夫君子者，惧一善之不力，则冥冥者有堕行，一不善之不去，则涓涓者无已时。屋漏而懔如帝天，方寸而坚如金石，独知之地，慎之又慎。

精英：

  1. 儒家的精英教育，不是伊顿、哈罗，而是从四书五经开始，反反复复唠叨的“修行”——言行学识，一步一个脚印，简单但不易行。
  2. 崇尚贵族是笃信遗传。
  3. 崇尚精英是笃信修行。所谓道理，都在这本薄薄的书里，是不是愿意修行，是不是修行到，看造化和虔诚。
  4. 成事理论的核心基础是：就是这么一个肉身，就在现世，还是可以努力做圣人，做不到圣人，至少可以做一个成事的人，至少可以让世界更美好一点点。

过错：

  1. 远古时候没有规则，都是草创期，人往往犯错而不自知；
  2. 儒家形成之后，什么是错是清楚的；
  3. 孔子尚且有错，现在的人怎么可能没错呢？
  4. 知道自己错了而不改，是奸；
  5. 知道朋友错了而不劝，是媚。
    ——又有多少人能简单、坦诚、阳光地和僚友们掰扯是非、毁誉和成败？掰扯之后，反复掰扯之后，又有几个人能简单、坦诚、阳光地接受和改善？

成事：先立志，再用心，最后坚持住。——学贵初有决定不移之志，中有勇猛精进之心，末有坚贞永固之力。

凡物之骤为之而遽成焉者，其器小也；物之一览而易尽者，其中无有也。

  1. 易做的事，都是小事；易读的书，都是简单清浅的书。
  2. 不要走捷径，捷径都是通向邪路最快的路。
    ——未选择的路

君子赴势甚钝，取道甚迂，德不苟成，业不苟名，艰难错迕，迟久而后进，铢而积，寸而累，及其成熟，则圣人之徒也。

  1. 走远路，下笨功夫，一点一滴进步。全力争取名实相符，尽量不要德不配位。
  2. 在现世，难。现世的常规是能捞一笔是一笔，能骗一时是一时，能忽悠一拨儿是一拨儿，成名要趁早。

如何不求人，保持独立？过简单生活。

安乐之时，不复好闻危苦之言。人情大抵然欤！君子之存心也，不敢造次忘艰苦之境，尤不敢狃于所习，自谓无虞。

  1. 享受的时候，不爱听创业时的苦，这是人性。
  2. 但是，生于忧患死于安乐，也是古训。
  3. 坚守常识，超越人性，就是精英。

陈丹青

陈丹青：

  1. 什么叫做救自己呢？就是忠实自己的感受，认真做每一件事，不要烦，不要放弃，不要敷衍。
  2. 真正有效的教育是自我教育。
  3. 真率是很高的要求。真率也是品德。

梁启超

莫问收获，但问耕耘

孩子们：

思成和思永同走一条路，将来互得联络观摩之益，真是最好没有了。思成来信问有用无用之别，这个问题很容易解答，试问唐开元、天宝间李白、杜甫与姚崇、宋璟比较，其贡献于国家者孰多？为中国文化史及全人类文化史起见，姚、宋之有无，算不得什么事，若没有了李、杜，试问历史减色多少呢？我也并不是要人人都做李、杜，不做姚、宋，要之，要各人自审其性之所近何如，人人发挥其个性之特长，以靖献于社会，人才经济莫过于此。思成所当自策厉者，惧不能为我国美术界作李、杜耳。如其能之，则开元、天宝间时局之小小安危，算什么呢？你还是保持这两三年来的态度，埋头埋脑做去了。

便对你觉得自己天才不能副你的理想，又觉得这几年专做呆板工夫，生怕会变成画匠。你有这种感觉，便是你的学问在这时期内将发生进步的特征，我听见倒喜欢极了。孟子说：“能与人规矩，不能使人巧。”凡学校所教与所学总不外规矩方面的事，若巧则要离了学校方能发见。规矩不过求巧的一种工具，然而终不能不以此为教，以此为学者，正以能巧之人，习熟规矩后，乃愈益其巧耳。不能巧者，依着规矩可以无大过。你的天才到底怎么样，我想你自己现在也未能测定，因为终日在师长指定的范围与条件内用功，还没有自由发掘自己性灵的余地。况且凡一位大文学家、大美术家之成就，常常还要许多环境与及附带学问的帮助。中国先辈说要“读万卷书，行万里路”。你两三年来蛰居于一个学校的图案室之小天地中，许多潜伏的机能如何便会发育出来，即如此次你到波士顿一趟，便发生许多刺激，区区波士顿算得什么，比起欧洲来真是“河伯”之与“海若”，若和自然界的崇高伟丽之美相比，那更不及万分之一了。然而令你触发者已经如此，将来你学成之后，常常找机会转变自己的环境，扩大自己的眼界和胸怀，到那时候或者天才会爆发出来，今尚非其时也。今在学校中只有把应学的规矩，尽量学足，不唯如此，将来到欧洲回中国，所有未学的规矩也还须补学，这种工作乃为一生历程所必须经过的，而且有天才的人绝不会因此而阻抑他的天才，你千万别要对此而生厌倦，一厌倦即退步矣。至于将来能否大成，大成到怎么程度，当然还是以天才为之分限。我生平最服膺曾文正两句话：“莫问收获，但问耕耘。”将来成就如何，现在想他则甚？着急他则甚？一面不可骄盈自慢，一面又不可怯弱自馁，尽自己能力做去，做到哪里是哪里，如此则可以无入而不自得，而于社会亦总有多少贡献。我一生学问得力专在此一点，我盼望你们都能应用我这点精神。
