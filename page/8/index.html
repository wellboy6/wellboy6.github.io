给毕业生的忠告

发表于 2017-05-27 | 分类于 转载

  1. 能力的提升比涨工资更重要，虽然你特别缺钱。别跟同事比薪水，「那个蠢货比我技术差，凭什么比我薪水高?」蠢货，比了也不会让你薪水立刻涨上来。

  2. 老板不是资本家，他/她也没有剥削你。这大概是你受的教育里最影响你职业发展的一个垃圾观念了，赶紧删除。

  3. 注意仪容仪表，尽管你是个工程师，但也不能穿得别太邋遢。

  4. 你的同事不是傻伯夷，你的客户也不是傻伯夷，你自己才是。所以，对他们出现的错误要能够理解。

  5. 在客户那里，你代表的就是公司。别跟客户抱怨公司的产品不给力，这实在太蠢了。

  6. 做好小事，处理好小事的细节。相信我，这时候你真不是做大事的料。

  7. 养成一个好习惯，比如每天做记录，每天回顾自己。跳槽后你会发现比很多人都牛。一个好习惯秒掉一大部分人。

  8. 从现在起，你认识的每个人以后跟你可能都有合作关系。争取让他们第一印象对你好一点。

Hi,long Time No See~

发表于 2017-05-25 | 分类于 随笔

最近经常性失眠，睡眠质量不高，躺在床上想的最多的一句就是：“Hi,long time no see~”。

好久不见，一个人的生活免不了孤单，也会经常性地感到寂寞，每次伤感的时候就想到药师博客的这句：好久不见。说给昨天的所有人。

开学第一天对宿舍只有一个感觉：陌生。寒假颠倒黑白的时间过去了，寒冷的一个冬天里，我终于积蓄了足够的能量让自己抵御心底的寒冷。我是太阳，我隐约感到尼采说这句话时候的心境。我是太阳，你靠近我，是想要得到足够的能量，你不是太阳。你离开，我还是要继续发光发热，我是太阳。一个太阳，应该寻找另一个太阳。一个冰冷的星球只会不断地掠夺你的热量。当你不再温暖的时候，你自己，还是要孤独地继续抵御寒冷。

一个人的季节，完全没必要去回忆曾经的痕迹。设想千万种相逢，也只是一句，好久不见。

静下来去创作

发表于 2017-05-25 | 分类于 随笔

这段时间状态不好，这两天感冒了，下午吃了点药，感觉好了点。期末考试慢慢迫近，自己还是没办法静下心来去复习，就像平时，无法静下心来去听课学习。昨天看了夕雾之前推荐的一部电影《考试过关的艺术》，电影开头对于死亡的论述让我想到了自己。我是在做些什么，我到底是想干些什么，心中有一个理想的状态，却总是回避那种状态。

一年过去了，心里完全没有时间的概念，这一年的一个个坐标都和爱情有关。去年元旦之后回家自己开了一个小站《新闻三十分》，是想每天都把当天的新闻整理出来，结果却一次也没有做到，很遗憾，如果坚持到现在，回顾一下，应该就有一种时光流逝的感觉。或许没有，《小众阅读》做了两个多月，回过去看看，感觉很麻木。

前天就想写今年的年终总结，突然蹦出的感觉又突然消逝，只能留下一个题目。刚才去北斗网逛了逛，没有去年这个时候的那种阅读欲望，那段时间自己心情低落时就去北斗转悠一圈，都会有所发现。那个时候自己的阅读空间只订阅了北斗一个网站，现在的订阅里满满地排了几十个独立博客和十几个媒体门户。我跟药师说，应该是选文的缘故，对不那么优秀的的文字失去了一种感触。药师说，不要老是端着，会很累，享受不到阅读的乐趣。

都已经一年没有写什么东西了，这一年，琐碎的日子里唯一真实的存在就是爱情。或许我在爱情里是一个弱者，祈求、担忧，不敢去诉说去感触那种敏感，反倒把敏感加倍留在了爱情的感触中。这不是一个好现象。

每周都要花时间去选几篇好点的文章放到小众阅读上，现在却越来越感觉自己选的文章没水平，自己都不满意。前天想要找到去年我在空间写的年终感触都要翻六七页才能看到2011年的记录。很可憎的事情，自己感觉很好的文章转载到空间没有人去关注；想要做一个平台分享好文章，压力之下却发现自己对文章的感触日渐下降。分享的快乐点逐渐走低。
