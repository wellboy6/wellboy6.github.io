黄药师：五封信

发表于 2017-05-28 | 分类于 转载

药师临行之前发布了五封信，有王佩、老妖、宋石男、王小峰，还有他的一篇唠叨，转载如下。

生活如此，一路顺风。

五封信之一

宋老师：

您好，很冒昧给您写这封信。百忙中不知道您是否有时间看到这封信。

不知道该怎么表述，即将离开大学校园，心情格外平静。

作为一个工科的学生，毕业之后马上就要进入煤矿部门工作。在此之前，学校的工作基本都清一色的洗脑性质和行政工作，基本没有关注内心的引导或者课程。或者每一个工科院校的学生，都有一颗麻木和机械的心。互联网重度使用两年，阅读你的作品，看你的博客，微博，以此来提高自己的内心的素养或者内心的厚度。无论理性还是感性，在您的博客里更多的学到的是思维清晰的看待问题，在这里向您表示敬佩和感谢。

但是在这个毕业季，冒昧的恳请您，能给与毕业进入社会的我一些忠告。

时间无法提前，我亦无法探知明天。不希望经验套牢自己，但求一些你的只言片语来警醒或者指导自己。现实中或许无法遇到那些能让自己心悦诚服的人，写这封邮件一是了结自己一个心愿，一是想希望洞察社会的人能够给与一些点拨。

最后祝您工作顺利，身体健康

再次向您表示感谢。

祝好

——学生：×××

给药师

发表于 2017-05-28 | 分类于 随笔

药师：

近来安好？

昨天和今天六点起床，去海里游泳，水凉风凉浪高，天很蓝云彩很好看。在水里的时候很放松，所有纠结的东西都被遗忘。不知道从何说起。我以为这就是生活。这就是我想要的生活，只是两年来，我一次次逃离这种生活，逃到自己的心理世界去玩无谓的心理游戏。初中时候买了一本书，叫《读大学究竟读什么》，好像太超前了；很久之前读过一篇文章，作者说大学没必要搞得太紧张，美好的生活就很不错。我一直不懂，不为未来着急，不为工作、名利所迷惑，我以为这就是美好的生活。不止一次，我早起的时候看到威海早晨的明媚都会心生顿悟的感觉，让我心情明朗，转瞬又陷入心灵的伤悲之中，在悲观中不可自拔。我不知道为什么，我不想努力，我想不劳而获。我愤恨、自卑。我不止一次在日记里承认这一点，也不止一次在日记中说，要改变，终究只是说说。前天只睡了一个小时，昨天很困，想忍住，就一直看老友记，看奥普拉的采访。主演钱德勒的演员戒毒成功，他说如果你自己不转变，别人说再多也是枉然。如此。自己认识不到问题的严重性。在这样下去我就毁掉了。从7月26日暑假开始，我就感觉自己在腐烂生锈，沉浸在无边无际的性幻想中，每每不能自拔。明知是毒药，饮鸩止渴，却还是选择去做。不是对与错，是想或者不想。我在想自己到底在干什么，我忘掉了自己，不是忘掉了自己曾经想要什么，是忘记了自己是什么。不止一次，我自己说某些话，做某些事的时候，我都有旁观自己的感觉。很荒谬。前天去图书馆读尼采读加缪，加缪说，人生是荒谬的，但是人要强大，要反抗，反抗到底。尼采说，不要管明天，今天才是真实。我隐约能听到这种声音，我畏惧害怕。生活的路，看得太清楚总是显得艰辛，大多人都羡慕阿甘，可以一直做一直做，但是可以做得很好。我是我，只能如此。我感觉孤单，这种孤单在去年一整年的异地恋中没有改变。现在一个人，还是感觉孤单。想找个人陪自己。或者去逃避这种孤单，不去正式，恐惧依然。我也不知道自己害怕写什么，或许是害怕自己不够优秀，而自己确实不够优秀。那么今天和明天呢，昨天呢，这是我一直想要弄懂却越思考越混乱的问题，这个问题从没有因为我的思考而有所改变。废话太多了。很多时候我有种感觉很好的感觉，生活不错，我不去考虑明天，仅此而已。

在这围城之中，或者围城之外，我不知道该说些什么。

祝一切顺利

你叫过我肖申克先生

结婚：碎片化的生活

发表于 2017-05-28 | 分类于 随笔

谢天谢地，我又能稍微这么安静地思考和记录了。感谢July!

灵感来源于交流，交流就是倾听和倾诉。而单纯的聊天，不过是简单的诉说，除了打发时间，毫无意义。

生活是碎片化的，纵向的碎片关于时间，横向的碎片关于空间。是碎片组成了整个生活，生活从来都不是一个整体。以前我也曾经天真地以为生活是一个统一的整体，所谓变化都可以控制。这是一种幼稚的想法，凡是变化都是不可控的，可以控制的并不能称之为变化。碎片化的生活使得变化得以持续，碎片的产生和流逝并不会因为人的想法儿改变，从当时来看，这种产生和消逝显得相当突兀，然而如果你突然跳出棋盘用局外人的姿态去观察，就发现，真是碎片的产生和消逝，使得生活成为活的生活；如果有谁的生活是一个统一的整体，毫无疑问，这样的生活徒有外表而没有生命和灵魂。有这么一句话：人总是要有一次说走就走的旅行。诸如“这样的旅行”等等的行为或者事件就是一种打破生活整体性的碎片。

城邦暴力团

发表于 2017-05-28 | 分类于 阅读 ， 书摘

1.我扭头望了望小五，见她正觑眯着一对眸子打量院子里的各种手植青菜，登时那眸子便滴得出盈盈满满的苍翠之色来。那是一个让我永世难忘的神情——她就那么水灵灵瞪着丰园极为寻常的青菜叶子，照说应该为孙小六的失踪而操着心。可是不，不是那种操心，你甚至不觉得她脑子里正在想着她弟弟。我看得出那神情——我已经二十岁了，她也二十岁了，二十岁的男生看二十岁的女生一眼能看出很多东西——她那神情里有很多东西，就是没有操心。我当时说不上来，日后见识的女人多了——比方说有一个叫红莲的——就知道她们在用那种水灵灵的瞳光似乎十分专注地看着什么，还外带叹一口大气的时刻，其实满心只有一个念头：说得文气绉绉一些，那念头就叫向往；说得简单平白，就是想着另一种生活，羡慕着自己永远也不可能成为的一种状态。

2.孙老虎警告小五：万一孙小六又没了，他就把她的屁股达成两半八。小五把话同我说了，我说人的屁股本来就是两半八，不信你摸摸我的。小五说你嘴贱。
我嘴是贱，可情思确实炽烈、真实又纯洁的。已经是二十岁的人了，我没亲过女孩子的嘴，也还没报过女孩子的腰；现在我成天想着这个。不管街头巷尾哪个女孩子多看我一眼，我就想他妈这是“有女怀春”，我总不好意思给她吉士诱之一下子——一般来说这只是空谈瞎想白做梦的程度。可眼前孙小五那神情大是不同——我怎么看怎么觉得她不像是替孙小六或者她那眼见要挨揍的屁股担心；我怎么看怎么觉得她像是等着我上前搂住她，说：“我带你一起走算了。你爸找不着我们，谁也找不着我们。我们就去过另一种生活。”（手录于2012.4.15-2012.4.16）

短评：张大春的文字功底不是吹的，玩弄文字玩弄博闻强识还真没几个人能玩的过张大春。这部现代武侠小说，上半部是一口气通读到底，下半部读着读着就崩了，实在跟不上张大春掉书袋的节奏。索性记下来的摘录也就这两段，当时是看在张大春对女子描写如此传神的份上给抄下来临摹的。事到如今，我自个一个人还是分不清梦里梦外，也该醒了，都二十一了，我的黄金时代。
