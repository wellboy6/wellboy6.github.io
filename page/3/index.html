今日：七月

发表于 2017-05-28 | 分类于 随笔

曾经有个博客，建了一个页面，每日记录自己的状态。后来博客无法访问了，好像忙着考试，也就没去管，应该是没有备份了。下面是在lofter的一些随记，整理下。（2017.5.28）

我是否还需要另一个明天来整理自己，或许不需要了。放下焦虑，放弃对过去的不舍。在向前奔跑的路上，在与生活战斗的过程中逐步找回过去一丝丝的强大。2014.7.13

如果说我曾经失去过什么，那也一定是我自己没有努力的原因。努力向错误的方向比不努力还可怕。2014.7.13

无力感和无能的力量有所不同，前者只是看山还是山，后者过了看山不是山到了看山还是山的地步。所谓无力，不过是一事无成后一种颓废感；无能的力量更多表现为一切都完成后依然无力改变些什么的沧桑。2014.7.13

错的东西无论如何都是错的

发表于 2017-05-28 | 分类于 随笔

人生不打折

发表于 2017-05-28 | 分类于 随笔
