心的强大：给自己

发表于 2017-05-26 | 分类于 随笔

这篇文章是2011年下半年，写给别人的。很多次想起当时自己说的话，现在来说，是写给别人但要自己看的一封信。

苏遇的民谣，暑假时间给你发送过一次的。这几天我一直在听。（QQ邮箱坑爹，没法转发了。你邮箱里应该还有，想听就搜索下邮件自己下载哦~）

特别是《秘密》。

此外，辽宁的一个朋友让我去看北斗网，的确在里面发现了不少好文章。不成熟不够理性，但是真实。北斗网：http://ibeidou.net/

我们所矛盾和恐惧的无非是今天和明天罢了。就算是恋爱了，今天不能够在一起，明天不一定在一起。更不知道明天会不会美好。未知让我们变得懦弱，甚至改变我们心底真实的想法。

我们说，现在挺好，我们又想明天更好。既然今天地域与时间的阻隔无法在一起，那今天就不再强求了。

但是你要记得你渴望的事情，我是说你渴望的属于你自己的事情。如果你真的很想学金融，渴望去学，就努力！不要得过且过，不要不在乎。更不要自卑，你要记住你曾经是第一，既然曾经在巅峰，就要相信自己无论在哪里都可以重登巅峰！这无关乎出身，这是一种对自己灵魂的信仰。不要懈怠，但是不要顺从地接受教育，教育是社会之需求，学习才是内心之渴望。高中时候我没有跟你说过这些，因为我不说你可以走的更远。现在你到达北京了，那我告诉你，是希望你能够学会独立思考，强大自己的内心。强大的内心才会支撑你走下去。

不想在这里表达我的感情。我们心底都明白，只是现实在这里，我们不是理想主义者，我们屈从。

我想说，四年后呢？我昨天晚上笨了，我害怕四年后自己一无所有一事无成。就算我聪明了一晚上，可是剩下的几十年我要怎么去跟你交代。

我应该跟你说过的，我初三就恋爱过。有些时间你就像曾经的我和她，所以我害怕了。不是你不够优秀，是你还不够强大。优秀从来都不等于强大。我想和你在一起，我说过要和你在一起，但是现在我不说了不是不想了，只是因为不能说。

我希望你变得强大！我希望寒假的时候，明年暑假的时候，四年后毕业的时候，看到一个强大的你，可以勇敢的面对生活。

有时我在想自己是为了什么。不是想要找个小旅馆脱了衣服就开始做爱。心与心的交流感觉会更美妙。等待就相当于暧昧，唯一不同的是，一个保持距离渴望明天、一个若即若离享受今日。

我谈到做爱，是因为我的心底还有欲望在作祟。就算我勇敢了聪明了，你发了神经同意了。一旦欲望超过感情，得不偿失。

有些时候我就在想，你会找男朋友吗。我也会观察周围的女生，哪一个追到手会更有面子。这就是生活，心底的矛盾还在，生活的欲望就继续。我一边希望跟你在一起，一边还在意你是不是找到了怀抱，一边还在生活中继续着生活的繁琐。庸俗就是生活，一面心怀理想，一面庸俗的生活。

我不是完人，甚至不完全算是一个好人，但我想说我是一个真实的人。

我跟你说这么多，不是要告诉你不要找男朋友要好好等待。也不是控诉你的拒绝对我有多大影响。

我不想你错过！不是错过我，是错过生活。如果遇到就抓住，这没有什么对和错，是一个女生对自己的责任。被拒绝是一件很痛苦的事情，但是相比高一到高二两年的煎熬来说，痛苦对我只能算是一种成长。

如果你享受现在的生活，或者对明天看得到的生活完全满意，就完全忽略我的这封邮件！每个人的选择不同，你有自己的选择，我选择清醒选择追寻，选择探索，你可以选择不！如果不，请忘记这封邮件。

我只是想告诉你，让自己变得强大，变得清醒。我希望四年后毕业的时候见到你，见到一个全新的强大的你，不单单是漂亮的成绩单，还有你的内心！内心的强大才是真的强大，强大而清醒的内心才可以让你勇敢的追随内心！

我只是想说，我昨晚不愿意去承诺一个不确定的明天。但是当你也拥有一颗强大的心，你我才会做出无悔的选择。生活是一个人的生活，爱情是两个人的爱情。

你和我不一样的，我不知道你对我到底是怎么样的感情，包括你说爱我，你说want to be with me的时候。我不知道你的想法，因为我从未走进你的内心，你也从未走进我的内心。高中的时候我规划自己的大学就是一个人的大学，因为我不认为简单靠感觉的恋爱是大学的恋爱，如果只是喜欢，那不叫恋爱。

我只是在想，你能否也像我一样去追寻一些不一样的东西。哪怕这些东西是异端思想。

还是刚刚那句，如果你有自己的生活，就忽略这封邮件。

我是想告诉你，四年后，或者暑假里，或者寒假里，或者说暑期打工或者暑期支教时候，我们遇见了或者在一起了，我们会发现彼此的心靠的更近了。

如果四年后，你我单身依旧，如果四年后，你我感觉依旧，明天的一切不是会更好吗!

昨天看到一句话，在爱情里，男人需要尊重，女人需要关爱。爱情是有阶级的，是有尊卑差异的。

但是，如果心与心都无法靠近，爱情到底还是不是爱情！

同样的，昨晚我很笨。但是笨也有笨的好处，如果四年里你遇见了抓住了。就不需要等到四年后了。不是吗?孤独的人是可耻的，但人总有孤独的时候，今天孤独的人心里相信明天会有灵魂的碰撞，这不叫孤独。

我昨晚上笨了吧~但是不是像你说的一样不够勇敢、不敢相信。最简单的说法就是这是最好的选择。

我真的不知道你心底的想法，我不知道是不是应该继续坚持。但是做出这样的选择无关乎对错。因为明天会因今天的这个选择会更好不是吗？

我不知道你在学校到底如何，学校的氛围到底怎样，或者说你是否闻到了学校的味道北京的味道。高中的时间我说，在学校学习不是接受教育也不是背课本，去体味学校的味道体味学校所在城市的味道。不要问自己到底是什么味道也不要回答别人。闻到了就是闻到了，闻到了，是说不出来了。

高中时间我应该表达过感觉对中国大学很失望，肤浅！连大学里面的恋爱都变得肤浅而功利。这只是我的看法，不是在蛊惑你。

但是大学肤浅，大学里的你可以变得深刻。我希望你能够变得深刻起来，深刻的人才清醒，清醒的人才强大。

刚刚说过，我真的不知道你想要的是什么。我从初中甚至小学时候就没有屈从过内心，我无法理解你这么乖的女孩想要的是什么。如果我所说的生活不是你想要的生活，还是那句，就忽略这封邮件吧、

后悔没有好好学习考试作文了，现在写起东西来都没有条理，担待点~

其实说到底还有一点，你还是个女孩子。小女孩。

成功的花，

人们只惊慕她现时的明艳！

然而当初她的芽儿，

浸透了奋斗的泪泉，

洒遍了牺牲的血雨。 ——冰心

中学时候好多东西都有他们真实的含义，教育愚弄了我们。是痛苦让人清醒和成长，苦难过后，才会惊艳。而痛苦和苦难不是勤奋。

此外，我不了解家庭对你的影响，所以我不说。

最后的最后，写了快一个半小时了，也应该给自己理下头绪了。

感情是真的，生活是存在的。选择就是选择。

你我都说现在挺好，那就让现在继续下去。

现在挺好是希望明天更好。如果四年后缘分来临，那还有什么好说的呢！

如果遇见了抓住了，记得说出来。

如果真的是我这笨人一厢情愿，就告诉我。不然我一个人天天傻啦吧唧的很丢人的。

如果说你也爱我，你也want to be with me,就告诉我，两厢情愿骨肉分离的悲喜剧不差咱们这一个。再说两情相悦不是说一定要在一起了。因为感觉无法决定生活，但是感觉决定内心。

告诉我真实的想法。

就想我昨晚的选择，是不能。我是男人，或者说明天我要是男人，我应该做男人该做的选择。

此外，跟前边的话有点矛盾。真谈个恋爱，天天不见面不能找个小旅馆脱光衣服做爱也是一件很悲剧的事情。

我不是一个乖男孩，我希望明天会成长成一个男人。

你是一个乖女孩，如果你渴望学金融就去学吧！努力吧，没什么可以阻挡你内心的渴望，除非你从未渴望。不要自卑，自信是对自己灵魂的信仰。相信自己独一无二的灵魂。真正NB的人都是不可被替代的人！

我希望你也可以想我一样看一些不一般的书，想一些不一般的问题。但是这条路相对是危险而且未知的，未知是主要的，危险，我压根就没在意的。所以如果愿意，就上路，如果不愿意，就继续现在的生活。生活没有质量标准，尊重自己的选择就好。

所以我想知道你真实的想法。

这是表白还是求爱，我自己都搞糊涂了。我说出自己的思绪和想法。就像是我既希望你过得好，有希望能和你在一起，同时我又不知道你到底对我是什么感觉。就是这样。

就是这样，再写下去就没玩没了了，在这学校的机房里用两个小时写这些东西，有些是直接的，有些是含蓄的。不再多说了，说的再多还是这些破事。

PS：天可怜见，别看完邮件回复我知道了。
