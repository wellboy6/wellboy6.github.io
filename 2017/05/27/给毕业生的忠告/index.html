给毕业生的忠告

发表于 2017-05-27 | 分类于 转载

  1. 能力的提升比涨工资更重要，虽然你特别缺钱。别跟同事比薪水，「那个蠢货比我技术差，凭什么比我薪水高?」蠢货，比了也不会让你薪水立刻涨上来。

  2. 老板不是资本家，他/她也没有剥削你。这大概是你受的教育里最影响你职业发展的一个垃圾观念了，赶紧删除。

  3. 注意仪容仪表，尽管你是个工程师，但也不能穿得别太邋遢。

  4. 你的同事不是傻伯夷，你的客户也不是傻伯夷，你自己才是。所以，对他们出现的错误要能够理解。

  5. 在客户那里，你代表的就是公司。别跟客户抱怨公司的产品不给力，这实在太蠢了。

  6. 做好小事，处理好小事的细节。相信我，这时候你真不是做大事的料。

  7. 养成一个好习惯，比如每天做记录，每天回顾自己。跳槽后你会发现比很多人都牛。一个好习惯秒掉一大部分人。

  8. 从现在起，你认识的每个人以后跟你可能都有合作关系。争取让他们第一印象对你好一点。

一般人，三十歲前都賺不到大錢。

三十歲後要賺大錢，前題是，要看你三十歲前「投資自己」了沒有？

你都不願投資自己，你的上司，你的員工，更不可能看好你，把他們珍貴的資源，投資在你身上。

若是如此，三十歲之後，你就會真切地體會到，「人生開始走下坡」的滋味。

你會後悔你省下來的「毛毛錢」。

但，千萬切記，錢花了，不一定就能賺回來，

旨在「投資自己」的花錢，才有回收的機會。

ps：

有人問道，什麼叫「投資自己」，我的意見如下：

學習未知的學問。鍛練欠缺的技能。見識陌生的世界。以及，結交比你更好的人。

更重要的，給自己犯錯的機會。

說到底，也就是一句話，犯錯。

人不犯錯，不能長進。

三十歲前，要把所有的荒唐，固執，勇敢，夢想，幼稚，非理性，無邏輯，熱血澎湃，一意孤行，不畏人言，一切一切可能的「瘋狂」給實現了，即使不能實現，就算慘敗了，你還能平地而起，還能打翻身仗。

三十歲之後，你就不是為自己活了，是為你的老板，為你的員工，為你老婆孩子，為你年邁的雙親而活。在我眼中看到的世界裏，三十歲之後，絕大多數的人，就開始沒有自己了。也許有些人還能「繼續犯錯」，但，種種羈跘會不請自來，躲都躲不掉。

重要的是，犯錯。

三十歲，是道坎，因為你知道有這道坎，所以，三十歲前的你，會「雖千萬人，而吾往矣」。一定要「雖千萬人，而吾往矣」。一定要撞牆。不撞牆，你的人生，會失去很多意義，有一天，你總會後悔。你會對年輕時的自己，失望……。via：yolfilm
