游戏篇

沉默的大多数：被忽视的力量

技术导向还是用户导向。互联网公司游戏公司技术是基础，但是用户是根本，他的产品就是用户，做出来的成品没人用的时候就不是产品，通过用户变现之后才是产品。技术公司不一样，技术公司里面技术可以直接变现。

云计算和未来，过去和现在和未来。

用户成型之后，产品才做出来定型，此时此刻才能考虑经营产品的事情，这个时候的产品不再是代码组成的应用，而是用户。

传统行业的潜力比新兴企业要大；科学，「基础科学」导向的企业比技术导向的企业的未来更加广阔。

与此同时，科学和技术的持续增长给社会的传统也会造成持续的挑战，这种挑战会转化为压力和动力。法制社会短时间内依然不会建立起来，事实上法制社会的建立并不一定对科学和技术的增长完全有益。法律天然具有稳定性和完备性，科学和技术的迅速迭代使其天然具有的「叛逆性」，对未来的方向没有束缚制度对科学和技术来说才是最好的制度。

更重要的，当我们在讨论法制的时候，我们不是在讨论未来而是在讨论过去，我们希望通过法制的建立使自己过去受过的损害得到补偿，更多赞同法制建立的人是希望能通过法制的建立把过去取得的利益固定下来。中产阶级、没有红顶的商人以及知识分子对此具有极其强烈的欲望。天下攘攘皆为利来天下熙熙皆为利去。对于在过去的改革中没有取得利益的人来说，法制的建立有利于阻止其继续受到伤害。这一点是极其确定的。但是，到了当下这个时代，我们必须承认，从来没有立法先行或者法律先于市场制定这一说法，社会极其迅速的变迁以及这种变迁对不同群体造成的利益和损失，都使得法制的建立是被动的，而非主动的，是在迎接挑战而非指引未来。这一点，在短时间内无法改变。

个人主义的优点就在于，在这种法制没有建立又继续建立的社会变迁中，为自己争取更大的利益，群里的大多数的个人财富的增加有利于推动社会法制化和规范化。

我们面临的不是明天会不会是黎明的问题，黎明就不会在明天来临；我们面临的，是明天会不会是黎明前的黑暗的问题。无论如何，在黑暗来临之前，积蓄一点力量，做无论如何都无法充足的准备，才能让明天可能到来的黑暗短一点，让黑暗后的黎明来的早一点。

这样一个环境下，战斗是唯一的选择，生命不息，战斗不止。我们唯一可以选择的不过是在战斗的间隙寻找精神上的逃亡，美国也好欧洲也好，民主也好自由也好，电影也好文学也好，都是我们在现实中战斗的筋疲力尽之后选择的精神上的逃亡。于我而言，文学就是我的逃亡。

超越时代的艺术从不会做出具体的指导，而科学和技术的进步也不会是全面的进步，必然是某一点的进步推到了科学和技术进步的多米诺骨牌。

那些销声匿迹的创业公司

点心解散，轻博客低沉，糗百的销声匿迹．段子手从微博向知乎转移，微信的持续增长．陌陌的上市．

今天还没到黎明前的黑暗，所以明天，黎明不会到来．社会的压力存在，但还不够大，社会就像一个特殊的弹簧，只有压到底，才会彻底反弹．而现在还在下压的过程，一边下压一边增长．白岩松说「做新闻的理想主义不仅仅是让世界变得更好，在大部分时候是如何让世界不变得更坏」．现在的问题是让下压缓慢一点，如果能缓和到压力放松更好，然后在缓和的过程中保持住社会的增长。

在线教育或者移动端的教育有两个出路，一是解决掉言传身教的问题，让言传身教融入到互联网之中；二就是走出一条特殊的道路，让在线教育的结果不输于传统教育，或者说超越受教育者的预期。

传统教育出去学习之外，氛围环境等等都给人施加影响，这种影响本身也是教育。
