给药师

发表于 2017-05-28 | 分类于 随笔

药师：

近来安好？

昨天和今天六点起床，去海里游泳，水凉风凉浪高，天很蓝云彩很好看。在水里的时候很放松，所有纠结的东西都被遗忘。不知道从何说起。我以为这就是生活。这就是我想要的生活，只是两年来，我一次次逃离这种生活，逃到自己的心理世界去玩无谓的心理游戏。初中时候买了一本书，叫《读大学究竟读什么》，好像太超前了；很久之前读过一篇文章，作者说大学没必要搞得太紧张，美好的生活就很不错。我一直不懂，不为未来着急，不为工作、名利所迷惑，我以为这就是美好的生活。不止一次，我早起的时候看到威海早晨的明媚都会心生顿悟的感觉，让我心情明朗，转瞬又陷入心灵的伤悲之中，在悲观中不可自拔。我不知道为什么，我不想努力，我想不劳而获。我愤恨、自卑。我不止一次在日记里承认这一点，也不止一次在日记中说，要改变，终究只是说说。前天只睡了一个小时，昨天很困，想忍住，就一直看老友记，看奥普拉的采访。主演钱德勒的演员戒毒成功，他说如果你自己不转变，别人说再多也是枉然。如此。自己认识不到问题的严重性。在这样下去我就毁掉了。从7月26日暑假开始，我就感觉自己在腐烂生锈，沉浸在无边无际的性幻想中，每每不能自拔。明知是毒药，饮鸩止渴，却还是选择去做。不是对与错，是想或者不想。我在想自己到底在干什么，我忘掉了自己，不是忘掉了自己曾经想要什么，是忘记了自己是什么。不止一次，我自己说某些话，做某些事的时候，我都有旁观自己的感觉。很荒谬。前天去图书馆读尼采读加缪，加缪说，人生是荒谬的，但是人要强大，要反抗，反抗到底。尼采说，不要管明天，今天才是真实。我隐约能听到这种声音，我畏惧害怕。生活的路，看得太清楚总是显得艰辛，大多人都羡慕阿甘，可以一直做一直做，但是可以做得很好。我是我，只能如此。我感觉孤单，这种孤单在去年一整年的异地恋中没有改变。现在一个人，还是感觉孤单。想找个人陪自己。或者去逃避这种孤单，不去正式，恐惧依然。我也不知道自己害怕写什么，或许是害怕自己不够优秀，而自己确实不够优秀。那么今天和明天呢，昨天呢，这是我一直想要弄懂却越思考越混乱的问题，这个问题从没有因为我的思考而有所改变。废话太多了。很多时候我有种感觉很好的感觉，生活不错，我不去考虑明天，仅此而已。

在这围城之中，或者围城之外，我不知道该说些什么。

祝一切顺利

你叫过我肖申克先生

2013/8/19

亚辉：

网上认识一年有余了，你我都变化很多，我似乎能在你身上看到我多年前的样子，但是又充满了很多不同。毕竟每个个体都有其独特的一面。

说实话，多年前我懵懂，无知，甚至一无是处，但是我一直对自己赞许的是，我能在充满未知的时候告诉自己，要调整，通过阅读或者交流来调整，自己不能解决，就通过他人调整。如此反复的几个回合，真的发现男人真的有“大姨夫”症状的存在。如是说。

我喜欢打字的感觉，也喜欢与人写信交流，不过局限于我的朋友，导师和爱人（暗恋多年，追求到手相爱至今），但是没有任何一个人能够给予你明确的答案，只有自己能给自己个大体的方向，至于规划人生，连岳说那是上帝才会干的事情，人只能把每天做好， 所以你所想的超前于你的生活，而你做的恰恰脱离于自身的生活。比如，大学，你独立寻找解决问题的路子找到没有，自学一个你感兴趣的学科的毅力和能力，你搞到手没有。一个你暗恋的妹子你勇敢追求没有。男人的运动：篮球和足球，或者其他挥洒汗水的运动你参与了没有？

自命清高和游戏人间都没有错，错得是自己不知道自己究竟处于哪种状态。我推荐给你看《和时间做朋友》，或许早已经忘记了吧。俗话说的好：太阳底下没有新鲜事，所谓的每日新闻，看起来新鲜，换个国度或者换个大洲，不过寻常事。只不过我们在这个大的局域网里玩的不亦乐乎，每天关心粮食和蔬菜，也关心民主和自由。想起来有时候充满了矛盾和无力，关注完一阵还得继续现实的生活：没有圆珠笔你得去买，饿了一天还得跑去食堂吃饭，网费没了网线插得再多仍旧无法参与这热情洋溢打鸡血的生活。现实和梦想其实很近，但又很远，需要自己去认识认知并去协调。

毕业至今，有几点体会与你分享吧。首先就是运气，很让人泄气是吗？这让人无法捉摸的东西竟然是我深刻的认知之一，它不在乎当事人的能力，身份或者性别，你在某一刻与世界上或者至少与国家的某个人充满了千丝万缕的关系，影响到你此时此地的境地。所以很多时候让我觉得明天充满了多种可能，至于什么形状取决于你是乐观还是悲观。然后就是，时间vs理想vs物质–其中时间指的是你的青春时光，理想包括你的事业，你的爱情或者亲情，物质赤裸裸的解释就是金钱。如果现在需要10万让你拿出来给你母亲买一栋钟情很久的房子，你能拿出来多少，你能借多少？你能花费多久的时间来实现这一目标，你需要自身什么本领来得到这10万，为了这点钱会多大程度上影响到你的理想。

生活就是一件系统的工程，或许你需要不断的是错才能不断领悟，或许你每一步都走的恰好，但是我希望，最后的两年，要多读书，多写字，多看法律，多考证，少些虚伪矫情，少些无病呻吟蜻蜓点水，把爱好深入并坚持下去，把专业做到中等水平即可。

祝好。笔不前驰。
黄药师

2013.8.19
