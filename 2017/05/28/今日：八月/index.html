我从未像父母那样为生活负重前行。

我埋怨让我不如意的一切，却未曾意识到是我自己将自己至于此种地步。

唔，回过头去看这几年，一片空白，不是吗。

失落了三年，更加强大的内心让我说，继续前行。

——2014.8.28（注：又失落了三年？不是的，只是你太过悲观了啊。2017.5.28）
