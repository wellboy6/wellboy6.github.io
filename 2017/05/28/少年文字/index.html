为你写的诗

我相信

三百行诗词歌赋

没说出的

也只是三个字

我相信

三千年人来又人去

所留下的

也只是一段恩怨情

我提起笔

心却不知所以

我写下一行诗

却从字里行间看到你

把你写到诗里

我未加标点

我希望从始一个字到末一个字

从不曾停顿

从开始到结尾

从不曾中断咏叹唏嘘

把你写到诗里

以为能够留住你

却寻不到你的影子

只留下一声声叹息

在空中飘然散去

把你写到诗里

欺骗自己你不曾忘记

字与字句与句都只是回忆

淡了的 散了的

留在诗里 祭奠过去

把你写到诗里

执着总是痛苦的回忆

从来也不愿放弃有你的过去

不想打扰你

却打乱了自己

重复着你的痕迹

把你写到诗里

一切也将会过去

回忆只是回忆

从来也不曾强求

我也将放手

让你离去

——2009.02.24（好友作）

无题诗

人非人，物非物。

旧地空成黯然处。

孤影怅然一声叹，

怎堪知，丝丝悲苦掩不住。

昔非昔，忆非忆。

未语泪先流满地。

徒然捞月人不归，

谁曾想，缕缕相思何处寄？

——2009年四月书信附（好友作）

注：暑假翻阅旧时笔记，还留存着几年前与朋友的通信，有刘浩，也有水草；信的内容或许多了些隐私，随信寄过来的诗歌倒是保留了当年的影子，伊人已逝，谨以此纪念。（2014.2.23）

注2：也是在这个寒假之后，我还记得，2014.3.8，开始玩魔兽，然后就开始了逃避-想要逃离逃避-痛苦抑郁-继续逃避，这样一个环节之中。是时候结束这样一个循环了。不要再只是想想，开始去做吧。（2017.5.28）
