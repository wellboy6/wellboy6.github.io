民谣推荐

发表于 2017-05-28 | 分类于 音乐

深夜无眠，推荐一下我喜欢的民谣。

去年年底时候，受到黄药师的影响，开始听李志的民谣，当时最喜欢的几首歌是《春末的南方城市》、《关于郑州的记忆》、《黑色信封》、《被禁忌的游戏》、《翁庆年的六英镑》还有《女神》。《 被禁忌的游戏》有一个现场版，歇斯底里，经常被我翻来覆去播放，只是那些演唱会都删掉了，忘记了是哪一年的跨年演唱会了。

这被禁忌的游戏，早已忘记的岁月

这被禁忌的游戏，一如既往的岁月

随风飘荡的我们，黑夜里寻找一点点欢愉

也是在李志跨年演唱会的视频里，听到了声音与玩具的《最美妙的旅行》，还有声音碎片的《把光芒洒向更开阔的地方》和《 优美的低于生活》。

相爱吧 终有一散的人们

你失去的不过是童贞

等时光用尽了青春

你早已优美地在大街上融化

之后在豆瓣溜达到声音碎片的小站呆了几天，记不清都听过哪些，撕心裂肺的声音，似乎要冲破这生活、击碎这生活、反抗这生活。

今年六月份期末考试，舍友一直用iPad播放逃跑计划的《夜空中最亮的星》；接下来就是一遍又一遍的单曲循环。

我宁愿所有痛苦都留在心底

也不愿忘记你的眼睛

药师这就要去长春了，一个没有网络的地方，我完全忘记了曾近没有网络的生活我是如何度过的，一遍一遍地走来走去、消磨时间，静不下来去读书。看到药师说“抖擞一下身上倦怠的体毛”蓦然想起和菜头曾经提到的“我腿毛飘飘，壮怀激烈”。我一直想接一句：“我站在迎风的山岗上”。今天早上去海里游泳回来，整理自己网银支付宝时候发现，两年来自己混来混去，还在无所谓的东西上花掉了接近两千块人民币，不由一阵心痛、一天心烦。六级挂掉了，差点忘记这档子事，怎么就挂掉了呢？我完全没有料到自己会过，也完全没有料到自己会挂掉。不会再考了，突然浪费精力而已，我实在是静不下来，便这样厮混吧。

无处话心事，搁笔，去睡。

——2013.8.22

黄药师：五封信
2001：太空漫游
