空虚

发表于 2017-05-28 | 分类于 随笔

孤独和寂寞是怎样的一种东西，让你难过，让你空虚，让你无所事事，活死人一般。人们总是需要同类的。

肉体和精神的关系是什么样子的，我还是无法了解。只是失恋之后，一个人，真的是好空虚啊。没有力量，或者说，没有力量的源泉。找不到一种动力，失去了感觉，就像舌头失去了味蕾。生活呢，到底什么样子的生活才是有意义的？

真的很难过，很难过。

当你以为你找到了一个归宿的时候，却突然又成了一个人，你还是如此难过。

我操他妈的！

你是孤独的，真的。人是孤独的，如果你原本就是一个人，你不算孤独。可是突然你又变成了一个人，那种刻骨的孤独就显现了出来。

无聊，孤独，寂寞，空虚，没有动力，没有欲望。

我这是怎么了？

——2013.7.23
