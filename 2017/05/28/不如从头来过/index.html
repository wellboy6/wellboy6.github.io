不如从头来过

发表于 2017-05-28 | 分类于 随笔

SQLserver自从安装之后就没好好用过，印象中只有配置时候连接成功过一次，之前以及之后总是出现各种各样的毛病，连带的VS2010也无法使用。双学位考完试后，卸载这两个东西和之前安装的一些杂七杂八的软件，结果卸载太过火，直接把windows玩崩了，崩的相当干脆。无奈，恢复出厂设置。

前段时间朋友推荐我听张国荣演唱会，看他的电影，包括上面这部《春光乍泄》，英文名《Happy Togetter》。电影里张国荣饰演的张国荣有一句台词：“不如我们从头来过”。让人不忍拒绝。想想这半年来，一直混混沌沌，处于“混”的状态，混了这半年，这两年。一直没有把中心放在自己身上，放在当下，而是不断揭开结痂的伤疤，作茧自缚。那，不如从头来过。

去年年末搭建在GAE上的博客很少更新，免费限额也总是不够，垃圾点击太多，不熟悉Micolog，不能好好维护。加上之前注册的那些Google账号，密码过于私密，不想再去登陆；加上寒假头脑发热注册的两个域名，已经闲置了半年，不如重新做起。计划如下：

1.http://www.shenc.cn/

搭建在RedHat的云计算平台OpenShift上面，平台对PHP无限制，根据目前的收费策略，不会像GAE一样对免费用户形成巨大冲击。架构使用WordPress。

以后博客不再使用wellboy.net，改成现在这个，随笔什么的都写在这里。

2.http://www.wellboy.net/

链接到GitHub提供的主页服务上，方便以后整理自己的代码。

3.http://www.wellboy.cn/

寒假答应蚁翔的事情一直没有做好，前段时间本来打算写个HTML+CSS的中文主页，结果因为jQuery不熟练做崩了，之前想的是放到GAE或者SourceForge上，后来由于种种原因，也是太懒，连续做了两个网站太累，就直接链接到了网易的轻博客LOFTER上，域名绑定正在审核，预计没有问题，以后照片可以放到LOFTER上。诚如老鬼所说，点点前途已经显现渺茫，LOFTER背靠大树好乘凉应该撑得下去。

前天重装系统，一怒之下把手机、iPad、kindle、MP4、电脑硬盘以及两个U盘全部格式化，所有信息删除，包括一年来积攒了接近300G的电影和视频。不如从头来过。至于之前网络上众多账号，也都一并不再使用，除非需要备份。于是重新注册了三个邮箱：wellboy#yeah.net，TyrionChang#Gmail.com，TyrionChang#hotmail.com。其他有需要的账号都通过这三个账号重新注册。至于QQ，能不装就不装吧，WebQQ差不多够用了。

装完系统之后整个人还是比较悠闲的，需要什么装什么，速度快了许多，无用的软件也都不再碍眼了。虚拟机里面装个Ubuntu，有什么东西都更方便。只是，我泱泱景德镇是无敌的。

前几天想学习Git，准备去Ubuntu控制台安装。显示Git-scm.com无法访问，只能重新配置一遍GoAgent；本想直接控制台用sudo命令安装，结果应该是服务器IP在GFW的黑名单里，找不到软件包；好不容易翻墙下了Git源码跑到控制台去编译安装，结果提示我缺少某个头文件，Google到一个解决方案，然后好像有事应为被墙无法安装；无奈，只能直接windows下安装Cygwin，索性国内游镜像服务器，速度不慢。

其实这只是昨晚噩梦的开始。

RedHat的OpenShift本身没有在黑名单里，但是建立的应用使用的rdcloud.com大量被屏蔽，如同GAE使用的二级域名appspot.com。导致使用PuTTYgen公钥生成之后无法使用WinSCP连接OpenShift的服务器，无奈在命令行窗口里面ping二级域名的IP，IP依然被屏蔽，无法使用。然后找来找去还是台湾同胞帮我解决了这个问题：通过SSH账户连接。然后就去Google免费的SSH账户，发现这年头共享SSH账户的网站基本都已经长草，得之不易。借助SSH才最终完成OpenShift上WordPress博客的搭建。

有几个经验分享一下，windows下使用PuTTYgen和WinSCP两个工具足够了，大部分都可以在图形化界面下完成；之前需要命令行绑定域名，现在OpenShift后台已经有了图形化的选项，非常方便。唯一需要更改代码的地方就是将默认的英文版WordPress升级成中文版的设置。也比较简单，可以自行Google教程。

相比较而言，到是配置GitHub主页最为方便，GitHub托管的代码可以在线编辑，无疑方便了许多。搞定之后直接选个模板即可。之后本来打算把wellboy.cn绑定到SourceForge上托管成静态网站，之后如果有机会做UnionCar中文页面就直接放到SF.NET上面，不用再为服务器操心，不过现在看来还不如直接放到GitHub上做成主页更加方便。吃过晚饭之后突然犯困，想到床上躺会。突然想到能不能帮到点点上，又因为老鬼说的点点前途渺茫，想改到网易的Lofter上，之前Lofter一直不能绑定域名，就一直没有关注，现在可以绑定域名了，就果断直接先绑定到Lofter上，避免域名闲置折价，花钱买的放着不好，总要慢慢来用。

不如从头来过，前途很好。抑郁了一整个学期，反反复复的烦躁暴戾，一个又一个轮回，陷入自我牢笼里面；突然跳出来看看，才知道自己应该做减法的。wellboy这个昵称怎么看都有些幼稚，申请新的Gmail邮箱时候，决定使用TyrionChang。Tyrion：提利昂·兰尼斯特。

不如从头来过，微博关注被我删的只剩下两个：朝闻道和蚁翔。他们都还在路上，我却一直怯于出发，一直到今天失去了出发的欲望，那就停下来吧。两天前烦躁到极点的时候写日记回忆这两年、这五年、这八年、这十年、这二十一年的点点滴滴，决定从头来过，意味着遗忘过去。离去是背叛，忘记过去亦是背叛。黎耀辉，你还记不记得何宝荣？

不如，从头来过。
