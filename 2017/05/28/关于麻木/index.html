关于麻木

发表于 2017-05-28 | 分类于 随笔

原文：近代史课上看视频，看到日军残杀中国同胞的画面，有不少同学竟然失声而笑！我很愤怒！你们笑什么？！好笑吗？！这也让我想到鲁迅先生在东京求学时看电影，当看到同样的画面，日学生连连叫好的情景。我们是怎么了？难道我们真的已经麻木了吗！麻木到连沉重的历史都成为我们的笑料？！

刚才看到有一个好友发了这么一个状态，吐槽一下，主要是讲点其他的，非反驳。

吐槽：

有人笑，是讽刺，”攘外必先安内“，外贼打跑了，家贼成了王霸（王八）。

绝佳的黑色幽默。

反右、1959-1961、文革。这些东西都是敲锣打鼓进行中，大鸣大放，鲁迅说的愚昧始终存在。打日本的时候，中国百姓敲锣打鼓，打国民党时候，中国百姓敲锣打鼓，解放西藏新疆的时候，中国百姓敲锣打鼓，分土地打地主时候中国百姓敲锣打鼓，反右派的时候中国人民敲锣打鼓，文革保卫毛主席时候中国人民敲锣打鼓，改革开放时候中国人民敲锣打鼓，薄熙来在重庆的时候中国人民敲锣打鼓，薄熙来被双规的时候，有些人，不也在心里敲锣打鼓吗。

愚昧的前提是不独立，不独立，不是外贼强加，是家贼施加。

感激主人给自己好处的奴隶是个好奴隶，这奴隶，能做的长久。

而奴隶无所谓麻木。

——2012-4-26 12:29
