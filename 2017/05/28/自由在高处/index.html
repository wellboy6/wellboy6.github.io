自由在高处

发表于 2017-05-28 | 分类于 阅读 ， 书摘

1.这样的时代，没有谁比谁更幸运，只有谁比谁更不幸。

2.领导的要求：评论可以写啊，但不能有观点。

短评：这就是现在的中国，腐朽一般的外壳侵蚀每个人的灵魂，丢却了思想，就丢却了属于人类的热情。

3.当有的年轻人像我感慨不知道将来做点什么时，我会给他们两个建议：如果不想浪费光阴的，要么静下心来读点书，要么去赚点钱。这两点对你将来都有用。

4.在一个广场上，人挤人，你不知道方向在哪里，但如果你站得高一点，看得远一点，就知道周遭的种种拥挤对你来说其实毫无意义。

5.乔布斯说：你须寻得所爱。

注：世界只存在一种成功：做到以自己的方式度过一生。

6.而我始终相信，一个努力拓展言论自由的人，一定不忘拓展接受信息的自由，因为二者密不可分。

短评：之前我一直以为信息过量也没事，只要能平等获取即可，网络使平等获取信息成为可能。然而逐渐发现，信息过量对于我们而言，其实还不如信息匮乏；在信息过量的情况下，我们更不容易专注，反倒是站在信息链顶端的阶层更容易获得本质的信息，从而攫取和维护自己的利益。即便如此，接受信息的自由与拓展言论自由同样重要。

7.真正让我们不自由的，是我们的迷失的内心，是我们只知道协调自己与周遭的关系，而忘了更要让自己的人生走向高地，走向世界的屋顶，尤其要走上肖申克监狱的屋顶。（后记）

短评：出发吧，一刻也不停止，向前向前，哪怕一直在路上，向着远方向着高处出发吧。

8.原谅我这一生不羁放纵爱自由。（Beyond《海阔天空》）

乡村建设理论——中国民族之前途

1.天下事，明白的人自是明白，不明白的人总是不明白，这又不能不让我长叹息。

2.欧西之士，今日已感到陷入漩涡，无法自拔之苦；而远东方面不引为前车之戒，反思效尤，其结果不得同出一辙乎？（26.1.13-申报-丕斐-美）

1.上世纪八十年代以来的中国，之所以取得举世瞩目的成就，与其说是因为政府做了什么，不如说是政府没做什么。（P50）——未记录哪本书

2.不过，宋朝的开放也使尚处于野蛮中的民族看到了文明，并间接导致了西方社会在文艺复兴时期的爆炸式成长。

3.中国实行的是苏联全权体制，这种体制的特点之一，是党权代替行政权，行政权代替议会。这样做的优越性是干好事效率高，弊端是干坏事效率也高。

4.西方社会都面临未来的黎明，而中国却时时处处面对遥远过去的黑夜。——《中国人的素质》明恩溥

5.一个社会对付腐败最重要的两个工具，一是透明的媒体，二是司法独立。如果这两个工具都是在官僚体制的控制之下，他们永远也不可能自砍手脚。

6.丁学良：如果把二十世纪的中国政治和比较政治放在广阔的背景上看，过去的二三十年，是中国的官僚化××和发展的二三十年，是中国社会全面官僚化的二三十年。（××两字看不清了）
