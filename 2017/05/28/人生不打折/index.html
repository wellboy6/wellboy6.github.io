人生不打折

发表于 2017-05-28 | 分类于 随笔

学校超市没有妙脆角，想弄个乌龟坐骑，跑去淘宝购买。看到淘宝上的淘金币等各种各样的打折优惠信息，突然意识到一个问题。人生不打折。就如同前几天看到马亲王在知乎上说去实体书店看到好书一定买下来，回头再去网上买，时间成本太高。链接没有找到，不再给出。

换一句话，不应该追求打折，而应该追求更多的资本。

大学四年时间，过去的三年，剩下的一年，时间成本太低，都被我浪费掉了，整个人没有得到提升；换句话说，资本没有提升，想要更为好的生活只能追求打折优惠。但是同样的，追求打折以为这时间成本的提高，意味着要为了打折付出相当多的经历时间和选择。而一旦有了资本则不然，可以选择购买可靠的品牌，即购买那些不靠打折营销的品牌，这样，虽然金钱成本上升，但是时间成本下降。成长中我太多浪费了自己的时间成本，原因就在于金钱不足。

所以，少年，不要在夕阳下的海滩上散布来怀念你的青春了。去追求资本吧，把时间换成金钱，而非用时间去寻找打折。

——2014.7.4

注：时间不会打折扣，人生更不会，生活，也不会。（2017.5.28）
