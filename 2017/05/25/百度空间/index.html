希望在这里可以找到属于我的朋友，陪我走过这年华！

发表于 2017-05-25 | 分类于 随笔

前天初六，我的十七岁过去了，已经到来的是我的十八岁。

时间很快，我想到07年那个冬天，那个我最快乐的季节，有那麽多人陪着我，很快乐。现在，在这烦闷的高二里，我又是一个人，走在校园里，学会了承担，学会了担当，学会了孤勇。学会了接受这个世界，无论如何。

在这里，我希望能够碰到一些朋友，和我一起行走在这个季节，陪我一起笑看年华老去，给我一个记忆、一个新世界。

不知该说些什么，日子一天天溜走，病了2个星期了，每次得病都让我平静，让我感觉时间清晰可触。真实而不空洞！2010年就这么悄然而至，我没有任何感觉，没有去感伤，只是感觉自己要清醒起来，走下去，不管这个时间如何！

希望在这里可以找到属于我的朋友，陪我走过这年华！
