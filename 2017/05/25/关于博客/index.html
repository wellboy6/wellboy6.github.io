关于博客（旧）

发表于 2017-05-25 | 分类于 随笔

作者：WellBoy

时间：2012.11.21

2011年来到了威海，好友夕雾说这个小城是一个文化荒漠。最初的半年自己锐气无限，现在不过刚过去一年半，锐气似乎消退了许多。想给自己在网络上找一个自留地，CSDN、博客园都有考虑过，最后还是决定直接做个独立博客好了。简单明快。这个博客应该是一个新的开始。

今年跟朋友一起做杂志《独立日》，上个月开始在点点做集体博客【小众阅读】，因为资金的问题只用了.tk的免费域名www.theink.tk。月初的时候突然被刺激到，就下定决心要做个独立博客。因为【小众阅读】的缘故，开始做了这个独立博客。也因为我做独立博客的缘故，药师也终于决定做一个独立博客当《独立日》的门户。域名很NB，nbthinker.com。这是一个好的开始。

1.关于域名

因为在学习C语言，了解了C语言的历史之后，在探针论坛做了一个The God C Team。所以最开始想的域名是跟C有关的，但是跟GOD和C都有关的域名要么被注册了，要么就不喜欢。最后决定用shenc.cn（shenc.com和shenc.net都被注册了），取【神C】的意思。

因为那段时间心里一直装着独立博客的事情，有次在路上走着的时候嘴里念叨着well.well.well，突然wellboy这个词就冒了出来。所以就想到了wellboy.cn这个域名。

问了下朋友的意见，虽然shenc比wellboy要短，不过还是都比较喜欢wellboy这个域名，所以就决定使用wellboy.cn（wellboy.com已经被注册，wellboy.net还未注册）。

选择.cn的域名主要是因为2010年互联网寒冬时候.cn禁止个人注册，今年.cn开放个人注册之后就一直想注册一个。今年6月份得知开放个人注册消息时zhangyahui.cn还没有被注册，这次去注册域名时候却被抢注了，估计一是因为叫这个名字的人太多了，二是因为姓名拼音的.cn域名比其他域名要便宜很多。

之后因为.cn的域名解析有种种限制，就直接又注册了wellboy.net，都是三年，也是给自己一个激励！虽然现在wellboy.cn已经可以正常解析了，还是决定使用wellboy.net，wellboy.cn就暂时冷藏，看以后的需要再说。

至于wellboy到底代表有什么含义，我真的没有考虑过。刘未鹏的mindhacks.cn可以说是取头脑风暴的含义，wellboy不管怎么解释都显得比较牵强。

虽然没有具体的含义，不过wellboy叫起来还是很有感觉的，简单。

2.博客定位

主要会写一些技术类文章，记录自己的学习历程。同时会把自己的生活感悟记录下来。

初步有想法多写一些系列文章，从网络上整理的内容也不会少，毕竟技术有限。不过我会保证原创内容占到足够的比例。

《关于我》和《关于博客》这两篇文章都受到第一个给我友情链接的独酌逸醉的影响，感谢独酌逸醉，博客还没有架构好就给了我友链。同样也是受到他的影响，有想法把音乐等其他内容解析到wellboy.cn的博客。当然现在还没有建构，暂时会把wellboy.net和wellboy.cn都解析到这个博客，主域名是www.wellboy.net，其他域名都会通过url解析到这个域名上。

3.博客风格

博客风格主要指博客主题和字体。博客是用基于Python语言的Micolog搭建的，因为技术所限，暂时不会对主题和字体进行修改，以后根据技术水平会不定期对博客进行小的改动。

4.友情链接

对友情链接的要求不高，有意交换友链邮件我就好。
