博客搭建手记（旧）

发表于 2017-05-25 | 分类于 随笔

独立博客架构完毕，以后主要就在独立博客里厮混了，有想法就在GAE和SAE上多建立几个网站就好了，这里，就留着吧！
我的博客：www.wellboy.net

本来这篇文章的名字是《博客搭建手记：利用GAE和micolog搭建博客》，不过草稿存了好几天都未完稿，发现除了搭建博客之外还有很多话要说，后面的副标题就被我舍弃了。这次建博，是一个新的开始。

11月5号在新网了wellboy.cn域名，然后在GAE上用Micolog上搭建了博客，结果发现.cn域名很难解析到GAE上，只能放弃。11月11号在新网注册wellboy.net，然后注册Google App for Business试用版，通过Google App Engine Launcher将Micolog上传到GAE。

11月15号通过DNSPoD把域名解析到GAE，至此，博客正式搭建完成。

中间少不了很纠结的事情，最开始因为域名解析不到GAE、想用SAE搭建wordpress，弄好之后发现.cn域名不备案不解析，加上GAE对.cn支持也很差，就注册了wellboy.net。后来写了一个《关于域名用于接入境外网站说明书》，域名解析才给解禁。这时候wellboy.net已经解析好了，就只能暂时将wellboy.cn冷藏。

期间也发生了很多事情，例如寻找windows下Emacs安装教程时，遇到了独酌逸醉，然后得到了博客的第一个友情链接。例如根据自己安装使用Code::Blocks和Microsoft Visual Studio 2010 Express的经验写了一篇教程《Windows下C&C++非盗版IDE搭建之Code::Blocks与Microsoft Visual C++ 2010 Express安装和基本设置》，因此获得了法客论坛的邀请码。事事顺心如意，这真的是一个好的开始。

利用GAE和Micolog搭建独立博客

A.注册域名。

如果你英文足够好，推荐在国外的域名注册商例如Go Daddy那里注册域名，服务好限制少。当然，如果你对.cn域名比较痴迷，就只能在国内注册了。国内两大域名注册商万网和新网可以自己选一家，万网被阿里巴巴收购了，相对服务要好一点，收费也会贵一点。推荐域名注册商进行优惠活动期间购买域名。

B.域名解析。

推荐使用国内的DNSPoD。国外的也有OpenDNS和谷歌的Google Public DNS，不过不推荐使用，主要是不大符合国情，你们懂的。

C.GAE注册

首先使用Google账号登陆GAE，地址：https://appengine.google.com/。登陆后新建应用即可。

然后下载Google App Engine Launcher和Python2.5。有些网络教程中说2.X的版本即可，但是一开始我下载Python2.7安装后Google App Engine Launcher无法使用，所以推荐安装Python2.5。

D.Micolog安装

Micolog是基于GAE的类似于wordpress的使用Python语言编写的开源博客程序。作者徐明。

下载Micolog源程序，解压后首先根据readme的指引进行修改，然后复制源程序到Google App Engine安装文件夹。然后使用Google App Engine Launcher添加上传至应用即可。

E.域名绑定

注册Google App for Education。一定不要注册Google App for Business！前者是免费的，且不需翻墙，后者免费使用一个月，并且有可能需要翻墙，看你运气了，所以一定要注册前者。注册时记得使用要绑定的域名。另外Google App套件不支持.cn的域名，虽然注册后可以添加，但是还是需要你首先拥有一个非.cn的域名。

注册完成后最好进行域名所有权验证。然后登陆GAE进入相应的应用，到应用设置里面添加域名即可，具体可以搜索网络教程。当然如果有机会我也会写一个关于Micolog的系列教程，暂时不会详细讲解了，毕竟网络上已经有足够的教程了。

域名绑定的帖子网络上也有不少，如果找不到解决方法也可以邮件我。

因为谷歌赠送给GAE应用的appspot.com二级域名全部被墙，所以就必须要绑定顶级域名了。这种情况下，也是可以使用SAE来搭建博客的。例如leo108的博客就是在SAE上利用wordpress搭建的。不过如果域名不备案，所要消耗的云豆（SAE使用的计费单位）是已备案域名的双倍。也是因为个人技术能力不够没法获得SAE的开发者认证，才最终决定注册.net域名并使用GAE的。

不过好的是，SAE支持PHP语言，并且逐渐开始支持Python以及JAVA。如果要架构网站，PHP语言无疑是最好的选择。
