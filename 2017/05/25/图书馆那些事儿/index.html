图书馆那些事儿

发表于 2017-05-25 | 分类于 随笔

大一在图书馆工作了一年，大二上学期一直在网络上沉沦堕落，一年半过去了，时间不多了，自己还没有真正和图书馆亲近过。前天跟药师聊天，说到寒假要干什么，药师跟我说：时间不多了，至少要学精一门语言。突然就让我想到，时间真的不多了，没有时间给你挥霍了。

很少来图书馆，每次看到一群人排队来图书馆自习，我心里都有一种莫名的排斥。一个个抱起了教科书抱起了复印的复习资料背着背着，书架上那么多书，被人遗忘。这样的一栋藏书楼，怎就成了考试前的自习室。后天就要回家了，明天计算机基础考试。看了看操作题还是不想去练了，裸考也是无所谓的事情。舍友叫我一起来图书馆，就背着笔记本来了。第一次，第一次把笔记本带到图书馆来上网。

记得印象中上次在图书馆静静读书是大一上学期复习的那段日子，舍友和我一起，大早上爬起来买个里脊饼，就跑到图书馆，他在背英语，我在看冯唐的《万物生长》。说不清那是什么一种感觉，一种本能的对于考试的排斥。

今年的考试也基本都结束了，大都回家了，没有回家的也在放松自己，考过就好。不甘心，明明不该是这个样子的。昨天在北斗看到《常识》对谌洪果的专访，很有感触。高中的时候，自己一直梦想能够改变高中的教育，哪怕一点点。最后，环境没有因我而变，我也没有因为环境改变。我还是感觉这是一件了不起的事情。

下午跑到三楼去找那本《我是一只IT小小鸟》，很有印象！大一在三楼工作的时候，一直都有一个习惯，一边上书整理书架，一边把感兴趣的书籍名字和索书号记下来。一年的时间，记下了有一两百本书的名字，虽然没有真正细读过多少，但后来在网络上看到别人推荐的书籍自己熟悉的时候，就有印象这本书在某排书架的某个位置，知道图书馆有这本书，找起来顺手不少。很感谢那一年的经历。

下学期有20门课，复习法学的时候，感觉高中那时候自己没有全力学习真是太对了，翻来覆去地背那六科课本，实在是一件够无聊的事情。四天四科考试，自己也不过花了四天的时间熟悉课本。考前有一个半月没有跟夕雾聊天了，在考场聊起的时候，夕雾告诉我，“我课本看了好几遍，感觉很多都有用，但是不考。我大学前就预料到的，大学成绩会低分飘过。前天跟柏拉图聊天，她说，思想和成绩不可兼得，我选择思想，成绩就必然会如此，我不后悔。”下午翻《我是一只IT小小鸟》的时候，有一位作者说，成绩和实践是两码事，考试不会考你太深入太底层的问题。面试就不一样了，别人不会看你拿了多少奖学金，只会看你写了多少高质量的代码。

学习、成绩，在中国教育里，实在是一件死对头。无怪李志每次考试后十名就忍不住退学了。

写这篇碎碎念的时候，在听李志的音乐，《被禁忌的游戏》。这被禁忌的游戏，一如既往的岁月。刘未鹏说，写博客要写真正思考的结果，我是不认同的，倒是独酌逸醉的文风我很认同，时不时写一篇“思绪飘零”，有些时候，就这么吐槽，是很不错的选择。不会所有人都是批评家，不会所有人都是文学家。不能因为喜欢王朔不喜欢郭敬明就一定要写王朔一样的文字。你不能喜欢老罗，就可劲吃变成某胖子可爱多。

这总是又一个开始，下学期，不能在宿舍呆的太久，慢慢，学会在图书馆安置下来，静下来看一些东西想一些东西，记录一些东西。不是所有的文字都必须是思考的结果，有些文字只是为了记录一个事实，或者一种状态。

时间不多了，总是要开始的！
