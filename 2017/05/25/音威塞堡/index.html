音威塞堡

发表于 2017-05-25 | 分类于 随笔

上周的东西拖拖拉拉到这个周末才舍得花出时间来整理，还真有够懒的。

今天在人人上看九把刀在北大演讲的文章——《人生就是不停的战斗》，看了《那些年我们一起追的女孩》的预告片；在预告片里看到九把刀睡觉的时候、沈佳宜在后面拿笔戳他的后背，我就想起高三那些日子。

可是那三年一去不回来。

高中时候很不安分！到了狠不安分的地步~

只是没想到到了大学自己可以这么安分。造化弄人，单身的男生你伤不起。

逛豆瓣的时候点开同城活动，就这么一个活动，没多想直接就参加了。回来之后感觉还是很好的，只是这个星期又发生了太多的事情，心绪像坐过山车一样，那感觉也早飞丢了。

记得强哥说，还好老天下雨了，不然人再多来点，我这地方可就坐不下喽~

呵！老天说不准还真是有眼的。

非常感谢Tina组织了这次沙龙，决定来威海之前，害怕威海会不会没有自己想要的味道。现在可以放下那颗心了！

威海很好，威海的气息我也很喜欢。我会用四年去体味威海的味道，还有山威的味道。

也感谢强哥的那颗经典1956的红塔山。一定抽时间去听乐队的演唱。

不知道前天神棍节的时候狂欢如何？单身的男生决定不纪念这坑爹的节日！

视频里面最让我感动的画面就是临近结尾的有关AtoZ的那一段。艺术家也许偏执，也许多愁善感，也许喜怒无常。但是艺术家的心底总有那么一片赤诚，如同婴儿一般的赤诚。

想起了间隔年中的蚁翔。上次关注你微博时候你还在香港做沙发客，不知道你现在到达新西兰了吗？有没有去搭一搭新西兰环海路上的顺风车？和女朋友是住在青年旅社还是新西兰沙发客的家中？

旅行总是有不同于旅游的感觉。

那三年就过去了，我往前看看还有四年的时间。这次沙龙是一个开始吧，你说威海是什么味道？我不知道。

我记得小石岛海风下分享的鱼鸟、荒丛中破败的人民公社、云层缝隙洒下的万张金光、还有那粼粼的海面；

我记得玛珈山向东那曲径通幽但是无比坑爹的小路，打死我也不相信北门的那几块破石头是上山的小路；

我记得哈工威的海好臭、鱼腥味好重、污染好严重，老天爷真看眼，国际浴场成了山威的西门海滩；

我记得环海路旁2千米的金沙滩（牌子上写的，没量过），沙龙结束后回来的路上，有个妹子（其实沙龙上的妹子全是我学姐）说：夏天的时候沙滩上就会有穿着小裤衩的俄罗斯老头和穿着比基尼的大姑娘。

保研路还是保研林?

沙龙结束后Tina直接带着威职的妹子敢末路公交了，我听从强哥的劝告赶上了Tina又决定跟新传的三个妹子（学姐）步行回校。

不用担心，是西南门不是保研地带~

结束了吗？上周五的事情而已，刚刚不过九天。

九天是可以发生很多事情的时间，有些时候，九天，比九年还长。

上午的时候听程恩富（中国社会科学院马克思主义研究院院长、其他略）的讲座，讲马经和西经，让我一个法律人情何以堪~还是默默期待贺卫方的到来。想了很多，北斗网上有篇文章的作者说，他的老师（他就读于香港中文大学)说：中国的政治改革需要六个月、经济改革需要六年、但是文化改革需要六十年。时间应该没记错，至少六十年不会记错的。

程恩富还提到前几天外交部请他去参加个会议，讨论社会主义好还是资本主义好。我擦~~~苏共刚垮台二十周年吗！

前天想要去TED网站上申请TEDx的license，结果注册账户时候country一栏里面豁然有Taiwan,一下子没了申请的欲望和激情。

TMD上周一的英语作文还没作，还是赶紧回去补作业吧！~
